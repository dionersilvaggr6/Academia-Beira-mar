# Research — Beira Mar

## Site de referência: 26fit.com.br (rede de academias, RS/SC/SP/PA)

Estrutura da homepage: Hero (carrossel) → "Porquê escolher" (4 pilares) → Planos (3 cards)
→ Localizador de unidades → App próprio → Institucional → Fala Connosco (formulário) → Footer.

Funcionalidades notáveis: CTAs "EU QUERO", localizador de unidades, WhatsApp flutuante,
formulário segmentado, links para app, loja externa.

## Reaproveitado para o Beira Mar
Hero+CTA, secção de diferenciais, cards de planos, secção de modalidades, WhatsApp flutuante,
formulário de lead, institucional + footer com redes. **Não** reaproveitado: localizador
multi-unidade (Beira Mar = 1 unidade), app próprio, loja externa (será F3 interna).
